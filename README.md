# 撮影データ受け渡し確認書

スマホで使える、撮影現場向けの署名・受け渡し確認アプリです。
日付、担当者、共有方法、データ内容、署名を入力し、白背景のPNG確認書として保存できます。

## 使い方

1. `index.html` をスマホやPCのブラウザで開きます。
2. 日付、担当者、共有方法、データ内容を選択します。
3. 署名欄に指やペンで署名します。
4. `PNG保存` を押すと、確認書画像が保存されます。

## 誰でも使えるように公開する方法

このアプリは静的ファイルだけで動きます。以下のファイルをそのままWebサーバーへアップロードすれば公開できます。

- `index.html`
- `styles.css`
- `app.js`
- `manifest.webmanifest`
- `service-worker.js`
- `.nojekyll`

## かんたんな公開方法

### Netlify

1. <https://app.netlify.com/drop> を開きます。
2. このフォルダ、または配布用ZIPをドラッグ&ドロップします。
3. 発行されたURLを共有します。

### GitHub Pages

1. GitHubで新しいリポジトリを作ります。
2. このフォルダ内のファイルをアップロードします。
3. `Settings` → `Pages` → `Deploy from a branch` を選びます。
4. `main` ブランチの `/root` を公開対象にします。
5. 発行されたURLを共有します。

## 補足

署名データは外部サーバーへ送信されません。PNG保存は利用者の端末上で行われます。
